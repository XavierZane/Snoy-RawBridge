# Snoy-RawBridge 使用文档

本文档面向最终用户，说明如何使用 Snoy-RawBridge 通过 USB 从索尼相机按需导入 RAW / JPEG 文件。

## 1. 使用前准备

请先确认：

- 手机支持 OTG / USB Host
- 使用的是可传输数据的 USB 线，而不是只充电的线
- 相机已经开机
- 手机和相机连接稳定

建议优先使用：

- 相机 USB 模式：MTP
- 手机端导入目录：默认 `Pictures/RAWBridge`

## 2. 首次使用流程

1. 用 USB 线把相机连接到手机
2. 打开 Snoy-RawBridge
3. 如果系统弹出 USB 授权提示，点击允许
4. App 检测到可浏览状态后，会显示连接成功提示
5. 首次进入时，先到配置页确认导入目录与 USB 模式
6. 保存完成后返回主页

## 3. 首页怎么用

首页就是图库页。

你可以在这里完成：

- 查看相机中的照片缩略图
- 按 `全部 / RAW / JPEG` 筛选
- 进入多选模式
- 全选当前筛选结果
- 导入所选照片

说明：

- 首页展示的是相机侧缩略图或内嵌预览
- 浏览时不会直接把所有原文件全部传到手机
- 原文件只会在你点击导入后开始传输

## 4. 多选与导入

### 进入多选

点击首页顶部右侧多选图标，即可进入多选模式。

进入后你可以：

- 逐张勾选
- 切到 `已选` 查看当前已选内容
- 点击 `全选当前` 选中当前筛选结果
- 再点一次 `全选当前` 可取消当前这一组的全选

### 开始导入

选好文件后，点击底部导入按钮。

导入过程中，底部状态区会显示：

- 当前导入文件名
- 当前进度
- 成功数量
- 失败数量
- 当前批次总数

如果需要中止，点击红色停止按钮即可停止当前导入。

## 5. 配置页说明

配置页用于设置导入行为。

当前主要设置项包括：

- USB 模式偏好
  - `MTP`
  - `PTP`
- 导入目录
- 按日期自动建目录
- RAW / JPEG 分开保存
- 断开后自动清理缩略图缓存
- 主题
  - 跟随系统
  - 浅色
  - 深色

另外配置页还提供：

- 关于
- 使用文档
- 分享

## 6. 历史页说明

历史页会记录已完成导入的历史记录，你可以看到：

- 时间
- 文件名
- 导入结果
- 保存路径

如果某个文件导入失败，也可以从这里回看失败记录。

## 7. 文件会保存到哪里

默认保存目录是：

```text
Pictures/RAWBridge
```

如果开启了按日期建目录，实际目录会按日期继续细分。  
如果开启了 RAW / JPEG 分开保存，RAW 和 JPEG 会进入不同子目录。

导入后的文件会通过 Android MediaStore 写入，因此通常可以在：

- 系统相册
- 文件管理器

中直接看到。

## 8. 缩略图缓存机制

为了让浏览更快，App 会把当前会话的缩略图先写入应用缓存目录。

这部分缓存：

- 只用于当前连接会话
- 不等于原文件导入
- 断开相机或停止会话后会自动清理

所以它不会长期占用手机空间。

## 9. 常见问题

### 9.1 为什么首页能看到图，但手机里还没有原文件

因为首页优先加载的是缩略图或 RAW 内嵌预览，不是完整原文件。

只有你勾选并点击导入后，原始文件才会传到手机。

### 9.2 为什么大卡时完整图库还是需要一点时间

虽然现在已经做了增量缩略图显示，但相机端仍然需要先枚举对象，卡里文件越多，完整列表出来的总时间也会更长。

### 9.3 为什么有些 RAW 没有正常预览

部分文件可能缺少可直接读取的缩略图或内嵌预览，这时会显示无预览占位，但不代表原文件无法导入。

### 9.4 为什么不能直接覆盖安装另一个版本

如果手机里装的是 debug 包，而你现在安装的是 release 包，或者两者签名不同，Android 会拒绝覆盖安装。  
这时需要先卸载旧版本，再安装新版本。

## 10. 推荐使用方式

更推荐你把 Snoy-RawBridge 当成一个“现场选片导入工具”来用：

- 接线
- 浏览缩略图
- 挑选需要的照片
- 只导入想要的 RAW / JPEG

而不是把它当成整卡备份工具。

这样会更符合这个 App 当前的一期定位，也能避免无意义地占用手机空间。
